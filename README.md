# Notes App

##*NOTES APP*

## Features of the App:-
1) Creating layouts.
2) Working with Recycler View.
3) Working with Sqlite Database.
4) Adding Notes to Database.
5) Updating existing Notes.
6) Deleting all Notes.
7) Swipe to delete a single note.
